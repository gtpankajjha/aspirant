This is a React-based User Management application that allows users to log in, view a list of users, and manage user details. The application fetches user data from the Reqres API.

Features:-

User Login
List of Users
User Details
Edit User Information
Create New User

Technologies Used
React: Frontend framework for building user interfaces.
Axios: For making HTTP requests.
React Router: For routing between different components.
CSS: For styling the components.
Installation
Clone the repository:


git clone <repository-url>
cd <project-directory>

Install dependencies:
npm install

To Start 
npm start
